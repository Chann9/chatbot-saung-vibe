# Requirement Chatbot AI USING GEMINI API KEY

## Background

Saya ingin membuat aplikasi chatbot menggunakan gemini api key, chat bot tersebut memiliki fitur sebagai berikut:
- Login dan register
- Role (admin dan user). untuk role admin, akan pertama kali dibuat oleh sistem dengan ketentuan berikut:
- Username: `admin`, password: `admin123`
- Chatbot detail
- send chat
- replay chat by AI
- Rename chat conversation
- Delete current chat
- Add typing indicator ketika AI mengetik jawaban
- Limit chat per masing-masing user (diset oleh admin, ada initial valuenya ketika user pertama kali daftar)
